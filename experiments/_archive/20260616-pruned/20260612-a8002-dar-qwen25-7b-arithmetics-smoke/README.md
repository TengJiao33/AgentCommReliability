## Observed Setup Issues

- A800_2 has local `Qwen2.5-1.5B`, but not `Qwen2.5-1.5B-Instruct`; the non-Instruct smoke produced repeated invalid text and empty parsed answers.
- Qwen2.5-7B-Instruct produced escaped braces around final answers in some debate responses; upstream arithmetic parser did not parse those before the compatibility patch.
- Upstream `--out_dir` did not control all output files before the local output-path patch.

## Caveats

- Two samples are not evidence of method performance.
- The current smoke verifies launch, local model mapping, basic debate, and `filter_critical` code paths only.
- `result/debate_logs.jsonl` and `result/token_logs.jsonl` are still global append logs in the remote clone.
