B        More Implement Details

In our main experiment, we construct each multi-agent and ensemble method by using only open-source
models to achieve reproduced and fair comparison. We follow (Wang et al., 2024b) and adopt: Qwen1.5-
72B-Chat (Bai et al., 2023), Qwen2-72B-instruct (Yang et al., 2024), WizardLM-8x22B (Xu et al., 2023),
dbrx-instruct (Team et al., 2024). Besides, we also introduce two more powerful closed-source models
from OpenAI: GPT-3.5 and GPT-4, for a thorough evaluation. For MoA and SMoA, we set 4 and 2 to the
processor number and layer number. For SMoA, we set the selection response number k to 2. We also
conduct further analysis on these hyper-parameters in Section 5. For MAD, we follow the original paper
and set the number of debating agents to be 2. For SC, we set the diverse reasoning path to be 4, which
leads to a comparable computational cost with our method. We use 0.7 as the temperature to sample more
diverse response from LLMs. All the experiments with open-source LLMs are based on together.ai1 ’s
corresponding APIs.
