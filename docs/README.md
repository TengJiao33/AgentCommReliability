## 新文档准入

新增文档前先问：

- 它是不是机器、资源、同步、目录管理或新实验协议？
- 它是否避免引用旧路线作为当前研究依据？
- 它是否能在没有旧实验上下文的情况下独立成立？

如果答案是否定的，不要把它放回 live `docs/`。
